## ğŸ“¦ Instalasi

### Prerequisites
Pastikan Anda telah menginstall:
- **PHP >= 8.2**
- **Composer**
- **Node.js >= 18**
- **MySQL >= 8.0**
- **Git**

### Step 1: Clone Repository
```bash
git clone https://github.com/RenggoPandora/simantesa.git
cd simantesa
```

### Step 2: Install Dependencies
```bash
# Install PHP dependencies
composer install

# Install JavaScript dependencies
npm install
```

### Step 3: Environment Configuration
```bash
# Copy environment file
cp .env.example .env

# Generate application key
php artisan key:generate
```

### Step 4: Database Setup
Edit file `.env` dan sesuaikan kredensial database:
```env
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=simantesa
DB_USERNAME=root
DB_PASSWORD=
```

Jalankan migration dan seeder:
```bash
# Fresh install dengan sample data
php artisan migrate:fresh --seed

# Atau migration saja
php artisan migrate
```

### Step 5: Storage Setup
```bash
# Create storage link
php artisan storage:link
```

### Step 6: Build Assets
```bash
# Development mode
npm run dev

# Production mode
npm run build
```

### Step 7: Run Application
```bash
# Start Laravel server
php artisan serve

# Application akan berjalan di http://127.0.0.1:8000
```
